git status|- mostra o estato da branch
git diff | mostra linhas alteradas (--staged)
git add . | adiciona todos os arquivos
git commit -m "commit" | faz um commit das suas alterações 
git log | mostra o histórico de commit
git restore .\arquivo | restaura um arquivo (--staged)
git push | envia commits para um repositório remoto
git pull | recebe commits de um repositório remoto
